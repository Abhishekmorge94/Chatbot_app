# 🤖 AI Chatbot Using Groq

An AI-powered chatbot application built using Python, Streamlit, and Groq API.  
This project provides a simple and interactive chat interface where users can communicate with an AI assistant in real time.

---

## ✨ Features
✅ Interactive chatbot interface using Streamlit  
✅ Real-time AI responses with streaming output  
✅ Conversation history support using session state  
✅ Integration with Groq API  
✅ Simple and user-friendly UI  
✅ Fast response generation  

---

## 🛠️ Technologies Used
- 🐍 Python
- 🎨 Streamlit
- 🤖 Groq API
- 🔐 dotenv
- 📦 os module

---

## ⚙️ How It Works
1. User enters a message in the chat input box.
2. The message is stored in session state.
3. The chatbot sends the conversation to the Groq API.
4. AI responses are streamed and displayed in real time.
5. Previous chat history is maintained during the session.
