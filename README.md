# 🤖 AI Chatbot using Groq + Streamlit

## 📌 Description
This is a simple AI chatbot built using Python, Streamlit, and Groq API.

## 🚀 Features
- Chat with AI
- Streamlit UI
- Streaming responses
- Chat history

## 🛠️ How to Run

pip install streamlit groq python-dotenv

streamlit run app.py

## 🔐 API Key
GROQ_API_KEY=your_api_key_here
